# prog

Внутренний портал AMMA Production для отображения графика занятости актеров.

## Хранение без сервера

Самый простой и надежный вариант для браузера — хранить данные в `localStorage`.
Портал сохраняет актеров и статусы сразу после изменения в ключ
`amma-portal-data`, поэтому при обновлении страницы все остается на месте.


Для локального просмотра достаточно открыть `index.html` в браузере.
