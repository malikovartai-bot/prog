# prog

Внутренний портал AMMA Production для отображения графика занятости актеров.

## База данных

Схема базы данных находится в `db/schema.sql` и рассчитана на SQLite. Таблицы:

- `actors` — список актеров и их роли.
- `availability_statuses` — справочник статусов (занят, свободен, под вопросом).
- `actor_availability` — календарь занятости по дням.

Схема поддерживает хранение и обновление данных, которые могут использоваться порталом.

## Интеграция

Фронтенд ожидает HTTP-точки сохранения:

- `POST /api/availability` — принимает `{ month, changes }` для сохранения статусов по дням.
- `POST /api/actors` — принимает `{ name, role }` для добавления нового актера.
