# prog

Внутренний портал AMMA Production для отображения графика занятости актеров.

## База данных

Схема базы данных находится в `db/schema.sql` и рассчитана на SQLite. Таблицы:

- `actors` — список актеров и их роли.
- `availability_statuses` — справочник статусов (занят, свободен, под вопросом).
- `actor_availability` — календарь занятости по дням.

## Локальное хранение без сервера

Портал использует SQLite прямо в браузере через `sql.js` и сохраняет данные в
`localStorage` (ключ `amma-portal-db`). Добавление актеров и изменение статусов
сохраняются сразу, без отдельного сервера.

Для локального просмотра достаточно открыть `index.html` в браузере.
